## 3. 模块明细

### 模块 A — 标题区
- 章节标识：`02 | 能源催化 / ENERGY`

- 副标题：以水电解析氧（OER）为场景，验证 AI 在真实科学问题中的假设生成与迭代优化能力

### 模块 C — 为什么重要 / 为什么难（4 项，每项含 图标 / 标题 / 说明）
| 标题 | 说明 |

| 绿色制氢核心技术 | OER 性能决定电解水制氢效率与成本 |

## 4. 内容关系要点（供重绘时保持逻辑）
1. **B 指标** 是成果证据，需视觉权重最高、最先被读到。
2. **C 痛点** 回答「为什么做这件事」，与 B 形成「难题 ↔ 成果」对照。
3. **D 闭环**（方法论）与 **F 设备流程**（硬件实现）是同一套 DRY→WET→READ→ITER 逻辑的两种表达，需保持术语一致、相互呼应。
4. **E 设备图** 是「干湿协同」中「湿实验」的实体落点，证明闭环可真实运行。
